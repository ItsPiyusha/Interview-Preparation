# Why use pos[a[i]] = i instead of insert?
* ```pos[a[i]] = i``` will overwrite an existing entry for ```a[i]``` if it repeats (often fine for Two Sum).
* ```insert({a[i], i})``` will not overwrite if the key already exists (it keeps the first index). Sometimes that’s what you want; often either works depending on duplicate-handling preference.
If you want the “don’t overwrite” behavior explicitly:
```cpp
pos.emplace(a[i], i); // or pos.insert({a[i], i});
```
