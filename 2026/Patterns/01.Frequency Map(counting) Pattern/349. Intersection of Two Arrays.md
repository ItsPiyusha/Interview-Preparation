Given two integer arrays nums1 and nums2, return an array of their intersection. Each element in the result must be **unique** and you may return the result in **any order.**
```cpp
class Solution {
public:
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        vector<int> intersection;
        unordered_map<int,int>freq;
        for (int x: nums1)freq[x]++;
        for (int y: nums2){
            if(freq.count(y)) intersection.push_back(y);
        }
        return intersection;
    }
};
```

Your current code returns duplicates if nums2 contains repeated values (and it can also return duplicates if nums1 has repeats), because once you find a value you keep pushing it again without “consuming” it or enforcing uniqueness.

```cpp
#include <vector>
#include <unordered_set>
using namespace std;

class Solution {
public:
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        unordered_set<int> s(nums1.begin(), nums1.end());
        unordered_set<int> out;

        for (int x : nums2) {
            if (s.count(x)) out.insert(x);
        }

        return vector<int>(out.begin(), out.end());
    }
};
```

```cpp
unordered_set<int> s(nums1.begin(), nums1.end());
```
* **What it creates:** an ```unordered_set<int>``` named ```s```.
An ```unordered_set``` stores **unique** integers (no duplicates), using a hash table.

* **What the parentheses mean:** this calls the ```unordered_set``` **range constructor**, giving it two iterators:

  * ```nums1.begin()``` → iterator to the first element of ```nums1```
  * ```nums1.end()``` → iterator past the last element of ```nums1```
* **What it does:** inserts every element from ```nums1``` into the set.
  * If nums1 has duplicates, the set keeps only **one copy** of each value.
* **Why it’s useful in intersection:** later you can check membership quickly:
  * ```s.count(x)``` (or ```s.find(x) != s.end()```) tells you if ```x``` exists in ```nums1```
  * Average time per check is **O(1).**
Example:

* ```nums1 = [1, 2, 2, 3]```
* ```s``` becomes ```{1, 2, 3}```
