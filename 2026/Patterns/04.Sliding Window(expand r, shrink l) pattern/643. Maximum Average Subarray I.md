LeetCode 643 — **Maximum Average Subarray I** (CPP)  
Link: https://leetcode.com/problems/maximum-average-subarray-i/

This is a **fixed-size sliding window** problem (window size = `k`).

## Idea (Sliding Window)
- Compute the sum of the **first `k` elements**.
- Then “slide” the window one step at a time:
  - Add the new element entering the window
  - Subtract the old element leaving the window
- Track the maximum window sum seen.
- Maximum average = `maxSum / k`.

Why this works: every subarray of length `k` differs from the previous one by exactly 2 elements (one removed, one added), so we can update the sum in **O(1)** per shift instead of re-summing `k` elements.

## C++ Solution
```cpp name=maximum_average_subarray_i.cpp
#include <vector>
#include <algorithm>
using namespace std;

class Solution {
public:
    double findMaxAverage(vector<int>& nums, int k) {
        long long windowSum = 0;

        // sum of first k elements
        for (int i = 0; i < k; i++) {
            windowSum += nums[i];
        }

        long long maxSum = windowSum;

        // slide the window from i = k to end
        for (int i = k; i < (int)nums.size(); i++) {
            windowSum += nums[i];        // add new element
            windowSum -= nums[i - k];    // remove element leaving window
            maxSum = max(maxSum, windowSum);
        }

        return (double)maxSum / k;
    }
};
```

## Complexity
- **Time:** `O(n)` (each element added/removed once)
- **Space:** `O(1)`
