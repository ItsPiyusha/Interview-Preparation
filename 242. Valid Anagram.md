1. If |s| != |t| → false
2. cnt[26] = {0}
3. For s do ++, for t do --
4. If any cnt[i] != 0 → false else true

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    bool isAnagram(string s, string t) {
        if (s.size() != t.size()) return false;

        array<int, 26> cnt{};              // IMPORTANT: initialize to 0

        for (char c : s) cnt[c - 'a']++;
        for (char c : t) cnt[c - 'a']--;

        for (int x : cnt) {
            if (x != 0) return false;
        }
        return true;
    }
};
```
This is the “count and compare” template.

Checklist:
* Did I initialize counts? ({} or fill)
* Did I handle easy early return? (length mismatch)
* Did I verify result and return? (loop through the cnt once.)

## Another Wrong Try
```cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        if (s.size() != t.size()) return false;
        array<int,26> cnt{};
        for (char c : s) cnt[c-'a']++;
        for (char c : t) cnt[c-'a']--;
        for (char c : cnt){
            if (cnt[c]>0)
                return false;
        }
        return true;
    }
};
```

* for (char c : cnt) is iterating the values in cnt, not indices.
* Then you do cnt[c], treating that value as an index (also char can be negative).
* Also the condition should be != 0, not > 0 (because counts can be negative too).
